## Choosing a Project

---
1. Stick to **What you know**

<i style={{color:`orange`}}>--------------</i>

- Contribute to Projects you're familiar with
- These maybe projects you use almost all the time

---

2. Search a Project by Language

<i style={{color:`orange`}}>--------------</i>


- GitHub has a search feature for you to find Projects
- Select a Project with a Programming Language you're comfortable with like Python or JavaScript

---

3. Check Trending List

<i style={{color:`orange`}}>--------------</i>

  - Working on Projects aligned with Current Market trends makes you look upto date.

---
