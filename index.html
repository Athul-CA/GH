Goto github.com and create an Account. After creating, Create a Repository by clicking the + symbol on the top right corner of GitHub.

@Athul-CA

Give the repository name as your_github_username.github.io
You should initialize without
the Readme(for this one's purpose only)

@Athul-CA

Let's Start by Adding you to Git on your Machine

Type in this command in your Terminal(without '$' symbol)

  • $ git config --global user.name "Your Github-UserName"
  • $ git config --global user.email "Your Github Email"
@Athul-CA

Now go to the Folder on your System with your Website's Files

Type $ cd "path of your workfiles"

@Athul-CA

Now we can Initialize the Repository with Git

Type $ git init

@Athul-CA

Now we need to add the Remote of the Repository

$ git remote add origin https://github.com/your_github_username.github.io.git This can also be found from your repository page on Github and you can copy from there.

@Athul-CA

Stage it!!!

Type $ git add -all
This will Stage all the files before Commiting

@Athul-CA

Commit it!!!

Type $ git commit -m "A Message"
This will commit all the files....

@Athul-CA

Push it!!!

Type $ git push -u origin master
You maybe asked to type in your Github id and Password.
Type those in and got to github.com

@Athul-CA

Is it here?

Refresh the page.... You will find all the files on your machine here....
