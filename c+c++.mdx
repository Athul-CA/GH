# History

---
## C

- Developed by Dennis Ritchie in Early 70's
- Made for the Unix Operatng System
- Procedural Design

---
## C++

- Developed by Bjarne Stroustrup in mid 80's
- C++ is **C with "classes"**
- Object Oriented Design

---

## Learning  C and C++

---

- C gives an easier API to provide access to low level access to the machine and memory.
- Useful for Systems Programming, Hardware Design, Understanding Programming Logic

---

- C++ is an Object Oriented Programming Language
- Complex API than C but gives same Powers
